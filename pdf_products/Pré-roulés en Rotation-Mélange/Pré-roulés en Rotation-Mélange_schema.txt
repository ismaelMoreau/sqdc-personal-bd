Voici un schéma de validation MongoDB basé sur les informations extraites des images fournies. Ce schéma est conçu pour capturer une variété de détails sur les produits de cannabis préroulés.

```json
{
  "type": "object",
  "properties": {
    "produit": {
      "type": "object",
      "properties": {
        "marque": { "type": "string" },
        "nom": { "type": "string" },
        "espèce": { "type": "string" },
        "dominance": { "type": "string" },
        "format": { "type": "string" },
        "description": { "type": "string" },
        "mélange": { "type": "string" },
        "effets_potentiels": { "type": "array", "items": { "type": "string" } },
        "temps_appartion_effets": { "type": "string" },
        "type_de_papier": { "type": "string" },
        "lieu_de_culture": { "type": "string" },
        "méthode_de_culture": { "type": "string" },
        "méthode_de_séchage": { "type": "string" },
        "variété": { "type": "string" },
        "génétique": { "type": "string" },
        "terpènes": { "type": "string" },
        "emballage_humidité": { "type": "boolean" },
        "sachet_humidité": { "type": "boolean" },
        "variations_couleur": { "type": "string" }
      },
      "required": [
        "marque",
        "nom",
        "espèce",
        "dominance",
        "format",
        "effets_potentiels",
        "temps_appartion_effets",
        "type_de_papier",
        "lieu_de_culture",
        "méthode_de_culture"
      ]
    }
  },
  "required": ["produit"]
}
```

### Explication des champs :
- **marque** : Nom de la marque du produit.
- **nom** : Nom spécifique du produit.
- **espèce** : Type de cannabis (Indica, Sativa, Hybride).
- **dominance** : Dominance génétique (ex. : Sativa, Indica).
- **format** : Détails sur le format (ex. : 5 x 0.5 g).
- **description** : Description du produit.
- **mélange** : Type de mélange (ex. : Blend).
- **effets_potentiels** : Liste des effets potentiels.
- **temps_appartion_effets** : Délai d'apparition des effets.
- **type_de_papier** : Type de papier utilisé pour les préroulés.
- **lieu_de_culture** : Localisation de la culture.
- **méthode_de_culture** : Méthode utilisée pour cultiver le cannabis.
- **méthode_de_séchage** : Méthode de séchage du produit.
- **variété** : Variété de cannabis.
- **génétique** : Informations sur la génétique.
- **terpènes** : Pourcentage de terpènes.
- **emballage_humidité** : Indique si le produit est emballé avec un contrôle d'humidité.
- **sachet_humidité** : Indique la présence d'un sachet d'humidité.
- **variations_couleur** : Variations de couleur de la cendre ou du produit.

Ce schéma peut être ajusté en fonction des besoins spécifiques de votre application ou de votre base de données.
